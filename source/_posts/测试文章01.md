# 算法

## 一、平方剩余和

> 主要用于快速的判断两个数的乘积是否为完全平方数。
> 如果平方剩余和相等，则两个数的乘积是完全平方数。

```java
import java.util.HashSet;
class Solution {
    // 平方剩余和 的预处理
    static int[] core = new int[1010];
    static int limit = 1000;
    static {
        // 求 1~1000 的所有数的平方剩余和
        for(int i = 1;i <= limit;i++) {
            if(core[i] == 0) {
                // 说明没有被计算过
                for(int j = 1;i*j*j <= limit;j++) {
                    core[i*j*j] = i;
                }
            }
        }
    }

    // 验证，平方剩余和 时，两个数的乘积为完全平方数
    public static void main(String[] args) {
        for(int i = 1;i <= 1000;i++) {
            for(int j = i + 1;j <= 1000;j++) {
                if(core[i] == core[j]) {
                    System.out.println((i * j) + " == " + (String.format("%.0f",Math.sqrt(i * j))) + "^2");
                }
            }
        }
    }
}
```
